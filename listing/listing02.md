Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
2
1


Это происходит потому, что в первой функции test() мы используем именованный возвращаемый параметр x,
который по умолчанию равен 0. Затем мы используем отложенную функцию для увеличения x на 1, но до этого
мы уже установили x = 1. Поэтому, когда мы возвращаем x, мы возвращаем 1, а затем отложенная функция
увеличивает его на 1, так что в итоге мы получаем 2.

Во второй функции anotherTest() мы не используем именованный возвращаемый параметр, поэтому мы создаем
новую переменную x внутри функции. Поэтому, когда мы возвращаем x, мы возвращаем 1, а отложенная функция
не имеет эффекта на значение x.

```
